# MARKDOWN
## 语法
### 1. 表格
- 基本格式

  |列1|列2|列3|
  |-|-|-|
  |数据1|数据2|数据3|
  |数据4|数据5|数据6|  

- 对齐方式
  |左对齐|居中对齐|右对齐|
  |:-|:-:|-:|
  |左|居中|右|

### 2. 链接/图片
- 链接：`[文本](链接地址)`
- 图片：`![文本](图片地址)`
- 链接中的图片：`[![图片文本](图片地址)](链接地址)`

## 其他
### markdown的转化
1. markdown转化为html
- 可以使用pandoc将markdown文件转化为html文件  
  示例：`pandoc markdown.md -o markdown.html`
